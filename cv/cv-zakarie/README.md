## 🚀 Utilisation

1. **Ouvrir le CV** : Double-cliquez sur `index.html` pour ouvrir dans votre navigateur
2. **Imprimer en PDF** : Utilisez Ctrl+P (ou Cmd+P sur Mac) puis "Enregistrer en PDF"
3. **Héberger** : Déployez l'intégralité du dossier sur un serveur web

## 📝 Points à personnaliser

Avant d'envoyer ton CV, remplaceç les trois emplacements dans `index.html` :

```html
<!-- Ligne 270 - Portfolio personnel -->
<p class="project-link">Lien : <a href="#">(À compléter : GitHub ou portfolio URL)</a></p>

<!-- Ligne 282 - Zil Lab -->
<p class="project-link">Lien : <a href="#">(À compléter : GitHub)</a></p>

<!-- Ligne 297 - UI/UX Figma -->
<p class="project-link">Lien : <a href="#">(À compléter : URL Figma ou portfolio)</a></p>
```

Remplace les `(À compléter : ...)` par tes vrais liens :
- **Portfolio** : `https://github.com/zakarie-djemai` ou ton URL personnelle
- **Zil Lab** : `https://github.com/zakarie-djemai/zil-lab`
- **Figma** : Lien public vers tes maquettes

## 📱 Responsive

- **Desktop** : Colonne gauche (35%) + colonne droite (65%)
- **Tablette (< 900px)** : Stack vertical, sidebar puis contenu
- **Mobile (< 600px)** : Full width avec padding réduit

## 🖨️ Impression

Appuyez sur **Ctrl+P** (ou **Cmd+P** sur Mac) pour ouvrir la fenêtre d'impression :
- Format : A4 ou Lettre
- Marges : 1.2cm (déjà configurées)
- Convertissez en PDF si besoin

Le CSS d'impression est optimisé pour un rendu noir et blanc professionnel.
